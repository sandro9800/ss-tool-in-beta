# CheatScanner

CheatScanner is a staff-focused tool that scans log files and `.jar` files to detect suspicious or prohibited cheats.

## Features

* Detects common cheats like **Doomsday**, **Meteor**, and **Raven B++**
* Identifies disguised/renamed mods
* Scans both logs and `.jar` files
* Attempts to detect **Vape** traces *(not fully reliable yet)*

## Status

🚧 **Beta Testing**
The tool is still in development, so bugs or missed detections may occur.
